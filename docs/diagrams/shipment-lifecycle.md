# Shipment / Container Lifecycle (Simplified)

## States
- PLANNED
- BOOKED
- CONTAINER_ASSIGNED
- GATED_IN
- LOADED_ON_VESSEL
- DEPARTED
- ARRIVED
- DISCHARGED
- HOLD
- GATED_OUT
- INVOICED
- CLOSED

## Allowed transitions
- PLANNED -> BOOKED (BOOKING_CREATED)
- BOOKED -> CONTAINER_ASSIGNED (CONTAINER_ASSIGNED)
- CONTAINER_ASSIGNED -> GATED_IN (GATE_IN)
- GATED_IN -> LOADED_ON_VESSEL (LOAD_ON_VESSEL)
- LOADED_ON_VESSEL -> DEPARTED (VESSEL_DEPARTED)
- DEPARTED -> ARRIVED (VESSEL_ARRIVED)
- ARRIVED -> DISCHARGED (DISCHARGED)
- DISCHARGED -> HOLD (HOLD_APPLIED) [optional]
- HOLD -> DISCHARGED (HOLD_RELEASED)
- DISCHARGED -> GATED_OUT (GATE_OUT)
- GATED_OUT -> INVOICED (INVOICE_SUBMITTED)
- INVOICED -> CLOSED (INVOICE_APPROVED)

## Rejection examples (invalid)
- GATE_IN before CONTAINER_ASSIGNED
- GATE_OUT before DISCHARGED
- INVOICE_APPROVED without INVOICE_SUBMITTED
